$$
\begin{aligned}
&\lim_{ x \to 0 } \dfrac{\sqrt{ 1+x\cos x }- \sqrt{ 1+x }}{x^{3}} \\[10pt]
=&\lim_{ x \to 0 } \dfrac{x(\cos x-1)}{\left(\sqrt{ 1+x\cos x }+\sqrt{ 1+x }\right)x^{3}}\\[8pt]
=&\lim_{ x \to 0 } \dfrac{x \cdot \left(-\dfrac{1}{2}x^{2}\right)}{2x^{3}}\\[15pt]
=&-\dfrac{1}{4}
\end{aligned}
$$
