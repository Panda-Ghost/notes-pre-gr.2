#定理 

设 $a_{n}\in \mathbb{R}, \; n=1,2,\dots$，设 $\{ a_{n} \}$ 有界

则 $\{ a_{n} \}$ 收敛当且仅当

$$
\begin{aligned}
\varlimsup_{ n \to +\infty }  a_{n}=\varliminf_{ n \to +\infty } a_{n}
\end{aligned}
$$

且当 $\{ a_{n} \}$ 收敛时，

$$
\begin{aligned}
\lim_{ n \to +\infty } a_{n}=\varliminf_{ n \to +\infty } a_{n}=\varlimsup_{ n \to +\infty } a_{n}
\end{aligned}
$$
